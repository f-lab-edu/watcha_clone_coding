# Button

범용 버튼 컴포넌트입니다. 다양한 스타일과 아이콘을 지원하며, HTML 버튼의 모든 속성을 사용할 수 있습니다.

## 주요 기능

- 🎨 완전히 커스터마이징 가능한 스타일
- 🖼️ 아이콘 지원 (텍스트와 함께 또는 단독으로)
- ♿ 접근성 지원 (aria-label 등)
- 🔒 비활성화 상태 지원

## 사용 예제

```tsx
import Button from '@/components/Button';

// 기본 사용
<Button>클릭하세요</Button>

// 아이콘과 함께
<Button icon="🎬">영화 보기</Button>

// 비활성화
<Button disabled>비활성화</Button>

// 커스텀 스타일
<Button 
  style={{ 
    background: '#007bff', 
    color: 'white',
    padding: '10px 20px' 
  }}
>
  커스텀 버튼
</Button>

// 이벤트 핸들러
<Button onClick={() => console.log('클릭!')}>
  클릭 이벤트
</Button>
```

## 접근성

- `aria-label`: 스크린 리더를 위한 레이블 제공
- `disabled`: 키보드 및 마우스 인터랙션 비활성화
- 기본 `type="button"`으로 설정되어 폼 제출 방지

## 참고사항

- 모든 HTML 버튼 속성을 지원합니다 (`onClick`, `className`, `style`, `aria-*` 등)
- 스타일이 포함되지 않은 순수 버튼이므로 프로젝트에 맞게 스타일을 적용해야 합니다
- 캐러셀 컴포넌트에서 네비게이션 버튼으로 사용됩니다
